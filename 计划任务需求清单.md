# 计划任务需求清单

## 1. 产品收益计算任务

### 任务名称
`ProductIncomeTask` - 产品收益生成

### 执行时间
每天凌晨 00:00

### 功能说明
- 查询所有进行中的订单
- 根据产品配置生成当日收益记录
- 收益状态设为"待领取"
- 24小时后自动过期

### 数据表
- `ntp_order` - 订单表
- `ntp_income` - 收益记录表

### 伪代码
```php
// 查询所有进行中的订单
$orders = Order::where('status', 'active')->select();

foreach ($orders as $order) {
    // 根据产品配置生成收益
    $income = [
        'user_id' => $order['user_id'],
        'order_id' => $order['id'],
        'amount' => $order['daily_income'],
        'type' => 'product',
        'status' => 'pending',
        'expire_time' => time() + 86400, // 24小时后过期
        'created_at' => time()
    ];
    
    Income::insert($income);
}
```

## 2. 收益过期任务

### 任务名称
`IncomeExpireTask` - 收益过期处理

### 执行时间
每小时执行一次

### 功能说明
- 查询所有待领取且已过期的收益
- 将状态改为"已过期"

### 数据表
- `ntp_income` - 收益记录表

### 伪代码
```php
Income::where('status', 'pending')
    ->where('expire_time', '<', time())
    ->update(['status' => 'expired']);
```

## 3. 邀请任务重置

### 任务名称
`InviteTaskResetTask` - 邀请任务周重置

### 执行时间
每周一凌晨 00:00

### 功能说明
- 重置所有用户的邀请任务进度
- 保留历史记录

### 数据表
- `ntp_invite_task_progress` - 任务进度表

### 伪代码
```php
// 清空本周进度
InviteTaskProgress::where('week', '<', date('W'))
    ->delete();

// 或者重置进度
InviteTaskProgress::update([
    'lv1_count' => 0,
    'lv2_count' => 0,
    'week' => date('W')
]);
```

## 4. 奖池开奖任务

### 任务名称
`PrizeDrawTask` - 奖池开奖

### 执行时间
每天早上 05:00

### 功能说明
- 统计昨日数据（LV1团队充值、个人充值、邀请人数）
- 选出前三名
- 发放奖励（1388、888、688 XAF）
- 记录获奖记录

### 数据表
- `ntp_user` - 用户表
- `ntp_recharge` - 充值表
- `ntp_prize_winner` - 获奖记录表

### 伪代码
```php
$yesterday = date('Y-m-d', strtotime('-1 day'));

// 条件1: LV1团队当天充值最高
$winner1 = User::where('team_recharge_today', '>', 0)
    ->orderBy('team_recharge_today', 'desc')
    ->limit(1)
    ->find();

// 条件2: 个人当天累计充值最高
$winner2 = Recharge::where('date', $yesterday)
    ->group('user_id')
    ->orderBy('sum(amount)', 'desc')
    ->limit(1)
    ->find();

// 条件3: 个人当天邀请人数最高
$winner3 = User::where('invite_count_today', '>', 0)
    ->orderBy('invite_count_today', 'desc')
    ->limit(1)
    ->find();

// 发放奖励并记录
```
