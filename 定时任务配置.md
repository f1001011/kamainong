# 定时任务配置说明

## 已创建的定时任务

### 1. 收益计算任务 (IncomeCalculate)
**功能**: 每日凌晨生成产品收益记录
**执行时间**: 每天 00:00
**命令**: `php think IncomeCalculate`

### 2. 收益过期处理 (IncomeExpire)
**功能**: 将24小时未领取的收益标记为过期
**执行时间**: 每小时执行一次
**命令**: `php think IncomeExpire`

### 3. 邀请任务检查 (InviteTaskCheck)
**功能**: 检查邀请任务完成情况并发放奖励
**执行时间**: 每小时执行一次
**命令**: `php think InviteTaskCheck`

---

## Linux Crontab 配置

```bash
# 编辑定时任务
crontab -e

# 添加以下内容
0 0 * * * cd /path/to/app_api && php think IncomeCalculate
0 * * * * cd /path/to/app_api && php think IncomeExpire
0 * * * * cd /path/to/app_api && php think InviteTaskCheck
```

## Windows 任务计划程序配置

1. 打开"任务计划程序"
2. 创建基本任务
3. 触发器设置为每天/每小时
4. 操作：启动程序
   - 程序：`php.exe`
   - 参数：`think IncomeCalculate`
   - 起始于：`D:\phpstudy\jiuge\kamainong\app_api`
