## 收益领取规则

### 时间规则
1. 购买时间：当天任何时间
2. 收益产生：次日凌晨00:00开始
3. 领取截止：次日23:59前必须手动领取
4. 保留期限：未领取保留24小时
5. 过期作废：超过24小时自动作废

### 领取示例
**场景**：3月17日23:00购买体验产品

| 时间 | 操作 |
|------|------|
| 3月17日 23:00 | 购买产品 |
| 3月18日 00:00+ | 产生收益，可领取 |
| 3月18日 23:59前 | 必须领取330 XAF |
| 3月19日 23:59前 | 未领取可补领 |
| 3月20日 00:00后 | 超时作废 |
