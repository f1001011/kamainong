## 一、项目概述

### 1.1 项目目标
开发一套喀麦隆地区的投资理财系统，包含：
- 前端移动端应用（Vue 3 + TypeScript）
- 后端API服务（ThinkPHP）
- 数据库（MySQL）

### 1.2 功能需求（来自 执行文档）
| 模块 | 功能 | 状态 |
|------|------|------|
| 产品体系 | 11个产品（1体验+10固定收益） | ✅ 后端完成 |
| VIP系统 | SVIP1-10等级，每日奖励 | ✅ 后端完成 |
| 收益系统 | 次日领取，24小时过期 | ⏳ 待完善 |
| 代理系统 | 7个代理级别 | ✅ 后端完成 |
| 月薪奖励 | 团队充值达标奖励 | ✅ 后端完成 |
| 奖池活动 | 每日奖池抽奖 | ✅ 后端完成 |
