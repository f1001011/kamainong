## 🔧 部署步骤

### 1. 导入数据库表
```bash
mysql -u root -p kamainong < invite_task_tables.sql
mysql -u root -p kamainong < invite_task_log_tables.sql
```

### 2. 配置定时任务（Linux）
```bash
crontab -e

# 添加以下内容
0 0 * * * cd /path/to/app_api && php think IncomeCalculate
0 * * * * cd /path/to/app_api && php think IncomeExpire
0 * * * * cd /path/to/app_api && php think InviteTaskCheck
```

### 3. 测试定时任务
```bash
cd app_api
php think IncomeCalculate
php think IncomeExpire
php think InviteTaskCheck
```
