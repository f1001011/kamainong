## 使用示例

```php
use app\model\Income;
use app\model\Order;

// 查询进行中的订单
$orders = Order::where('status', Order::STATUS_ACTIVE)->select();

// 查询待领取的收益
$incomes = Income::where('status', Income::STATUS_WAITING)->select();

// 获取状态文本
$statusText = Order::getStatusText(Order::STATUS_ACTIVE); // 输出: 进行中
```

---

## 注意事项

1. **不要使用魔法数字**：始终使用常量而不是硬编码的数字
2. **统一管理**：所有状态值集中在 `app/consts/Status.php` 文件中
3. **Model中引用**：Model中使用常量，方便调用
4. **前端同步**：前端也需要定义相同的状态常量，保持一致
