## ADDED Requirements

### Requirement: System provides global configuration
The system SHALL return global settings for client use.

#### Scenario: Get global config
- **WHEN** client requests global config
- **THEN** system returns min recharge, withdraw rules, service hours

### Requirement: System provides multi-language texts
The system SHALL support French language texts.

#### Scenario: Get French texts
- **WHEN** client requests texts
- **THEN** system returns all UI texts in French
