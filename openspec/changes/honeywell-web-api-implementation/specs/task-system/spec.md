## ADDED Requirements

### Requirement: User can complete invite tasks
The system SHALL reward users for inviting team members.

#### Scenario: Complete LV2 5-person task
- **WHEN** user invites 5 LV2 members who recharged
- **THEN** system awards 300 XAF

### Requirement: Tasks reset weekly
The system SHALL reset task progress every week.

#### Scenario: Weekly reset
- **WHEN** new week starts
- **THEN** system resets all task progress to zero
