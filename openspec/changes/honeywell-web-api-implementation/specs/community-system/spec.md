## ADDED Requirements

### Requirement: User can post withdraw proof
The system SHALL allow users to share successful withdraw proof.

#### Scenario: Create post with proof
- **WHEN** user uploads withdraw proof image
- **THEN** system creates post visible to community

### Requirement: User can like and comment
The system SHALL allow users to interact with posts.

#### Scenario: Like a post
- **WHEN** user likes a post
- **THEN** system increments like count

#### Scenario: Comment on post
- **WHEN** user comments on post
- **THEN** system adds comment to post
