## ADDED Requirements

### Requirement: Home page component
系统 SHALL 提供首页组件，展示产品列表和导航。

#### Scenario: Display product grid
- **WHEN** 用户访问首页
- **THEN** 系统显示产品网格和 CTA 按钮

### Requirement: Profile page component
系统 SHALL 提供个人中心页面，展示用户信息和菜单。

#### Scenario: Display user info
- **WHEN** 用户访问个人中心
- **THEN** 系统显示头像、余额和菜单列表

### Requirement: Investment page component
系统 SHALL 提供投资页面，展示投资列表和状态。

#### Scenario: Display active investments
- **WHEN** 用户查看投资页面
- **THEN** 系统显示进行中和历史投资

### Requirement: VIP activity page component
系统 SHALL 提供 VIP 活动页面，展示会员等级和权益。

#### Scenario: Display VIP levels
- **WHEN** 用户访问 VIP 页面
- **THEN** 系统显示所有 VIP 等级卡片
