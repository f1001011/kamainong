# AVIVA 喀麦隆投资系统开发计划

## 一、技术栈

**前端**: my_app (Vue 3 + TypeScript + Vite)
**后端**: api_php (ThinkPHP)
**数据库**: MySQL (已完成)
**参考设计**: https://lles-co.com/

## 二、开发模块

### 前端模块 (my_app)
1. 首页 - 投资产品展示
2. 登录/注册
3. 产品列表
4. 产品详情
5. 我的投资
6. VIP会员中心
7. 收益记录
8. 个人中心

### 后端API (api_php)
1. 用户认证 (登录/注册)
2. 产品管理
3. 订单管理
4. VIP管理
5. 收益管理
6. 代理管理
7. 奖池管理

## 三、开发顺序

**第一阶段**: 核心功能
- 用户登录/注册 API + 前端
- 产品列表 API + 前端
- 产品购买 API + 前端

**第二阶段**: 会员功能
- VIP系统
- 收益领取
- 个人中心

**第三阶段**: 高级功能
- 代理系统
- 月薪奖励
- 奖池活动
