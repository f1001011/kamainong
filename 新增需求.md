## 需求1：注册时处理上级关系

### 当前问题
- 注册时没有处理上级ID（agent_id）
- 没有自动填充 agent_id_1, agent_id_2, agent_id_3
- 没有生成团队号（user_team）

### 解决方案
```php
// 注册时接收上级ID
$agentId = input('agent_id', 0);

if ($agentId > 0) {
    // 查询上级信息
    $parent = User::find($agentId);
    if ($parent) {
        $agent_id_1 = $agentId;
        $agent_id_2 = $parent['agent_id_1'];
        $agent_id_3 = $parent['agent_id_2'];
        $user_team = $parent['user_team'];
    }
} else {
    // 无上级，生成唯一团队号
    $user_team = time() . rand(1000, 9999);
}
```
